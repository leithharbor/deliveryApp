## 😎 프로젝트명
투데잇츠

## ✨프로젝트 소개
쿠팡잇츠를 모티브로 구현한 배달 어플리케이션입니다. 판매자로 회원가입하고 자신만의 가게를 만들 수 있습니다. 여러 메뉴를 만들어 판매를 개시할 수 있습니다. 구매자로 회원가입하고 다양한 메뉴들을 주문하고 리뷰를 남길 수 있습니다. 



## 📆프로젝트 기간
2025-01-07 ~ 2025-01-13

## 🙋‍♂️팀원 소개
